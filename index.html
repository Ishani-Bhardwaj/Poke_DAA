About Us

We are second-year B.Tech students passionate about Data Analysis and Algorithms! We built PokéDAA Arena to make learning core DAA concepts fun and interactive. By combining a classic Pokémon theme with a custom C++ Backend, we wanted to create a mathematically rigorous but highly engaging way to test Graph Traversals and Greedy Strategies.

How to Play

  • Level 1 (DSA): Prove your knowledge! Pass the Data Structures MCQ by getting >=7 score to enter the Arena.
  • Shortest Path to the Pokemon: Guide Red to Pikachu. Move one tile at a time. Go through the grid using the shortest path to reach Pikachu for next level
  • Level 2 (Advanced Thoery): Solve some tough questions to keep going on in the game. You still have to score>=7!!
  • (Fractional Knapsack): You have a 50kg bag. Pick the berries with the best value-to-weight ratio to maximize your HP.
  • Level 3 (Final Boss): Code a question on the Indigo Plateau to finally finish the game..
